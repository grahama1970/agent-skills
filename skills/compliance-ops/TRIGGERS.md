# Compliance-Ops Triggers

The compliance-ops skill is invoked when the user mentions any of these keywords or phrases:

## Primary Triggers
- compliance
- compliance-ops
- compliance check
- compliance scan
- audit

## Framework-Specific Triggers
- soc2
- soc 2
- soc2 type ii
- gdpr
- hipaa
- pci-dss
- pci dss

## Activity Triggers
- compliance audit
- security compliance
- regulatory compliance
- compliance report
- generate compliance report

## Control Triggers
- access control check
- encryption check
- logging audit
- change management

## Example Usage
```
User: "Run a SOC2 compliance check"
User: "Check GDPR compliance for this project"
User: "Generate a compliance report"
User: "Audit this codebase for compliance"
```
